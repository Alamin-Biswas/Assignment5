What is the document object Model

DOM(Document Object Model) is a system of JS language by which programmer can interact with the HTML page . It allows programmer to modify html file using js and make the web page dynamic

How do you select an element from the dom

An element can be select in different way . there have some DOM selection system is js using this programmer can select any ID,Class, or element . For example , getElementByID(), querySelector() , etc.

What is event delegation in the context of the dom and why it is useful?

Event delegation হচ্ছে JavaScript-এ একটি কৌশল যেখানে ইভেন্ট শ্রোতাদের (event listeners) একাধিক child elements-এ আলাদাভাবে সংযুক্ত না করে parent element-এ সংযুক্ত করা হয়

How Do you Manipulate an element and style using the dom.

There have 3 step to modify or manipulate and element
1. Select the element by eny of the dom selector
2. Modify or style the element using property and value
3. Apply the changes to the DOM using methods like append(), insertBefore(), etc.
